# 🧊 NKU-3D-Vision

Official repository for 3D vision research projects from the Media Computing Lab, Nankai University.

This repository collects code, data preparation scripts, evaluation tools, and project documentation for research on 3D vision, point cloud understanding, visual localization, and vision-language reasoning in 3D scenes. Each research project is maintained in its own folder with a dedicated README for installation, data preparation, training, inference, evaluation, and citation details.
